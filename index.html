html布局小技巧

之前一直在浏览别人写的blog，从没有想过自己有一天也会来写下blog，今天试着弄下也没有什么好发表的，还希望各位见谅。。。 我先总结下我在做项目的时候遇到的关于布局中margin的应用。

删除侧栏最后的Li的下划线

css

.margin-bt{position: relative;background:#009900;overflow: hidden;} .margin-bt ul{padding: 0;margin: 0;} .margin-bt li{height:25px;line-height:25px;padding:5px;border-bottom:1px dotted #d5d5d5;margin-bottom:-1px;}

html

`

        <h2>这里是侧栏最后的Li还有下划线吗</h2>

        <ul>

            <li>侧栏1</li>

            <li>侧栏2</li>

            <li>侧栏3</li>

            <li>侧栏4</li>

        </ul>

    </div>

`

消除右边的空隙布局

css

li{list-style: none;} .clearfix:after{ display: block;height: 0;clear: both;content:"."; visibility: hidden;} .main-con{position: relative;width: 100%;} .main-con .main{width: 800px;margin: 0 auto;background:#0062bd;padding-bottom: 100px;}

html `

        <h2>这里是个水平线哈哈右边的空隙没有啦</h2>

        <ul class="main-ul clearfix">

            <li>1</li>

            <li>2</li>

            <li>3</li>

            <li>4</li>

            <li>5</li>

            <li>6</li>

        </ul>

    </div>

`

相对夫级模块水平居中

css .margin-md{position: relative;width: 800Px;height: 400px;background: #458fce;margin-top: 60px;} .margin-md .mid-con{position: absolute;top: 50%;left: 50%;width: 300px;height: 200px;margin-left: -150px;margin-top: -100px;background: #ff6600;color: #fff;}

html ` 

        <h2>快来看看有没有居中？</h2>

        <div class="mid-con">

            1111112222332

        </div>

    </div>
